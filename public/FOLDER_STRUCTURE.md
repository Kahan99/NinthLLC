# IMAGE FOLDER STRUCTURE

The project now uses one image/media folder only.

```text
public/
  images/
  IMAGE_UPLOAD_GUIDE.md
  UPLOAD_CHECKLIST.txt
  COMPONENT_IMAGE_MAPPING.md
  QUICK_START.txt
```

## Rules

1. Put all image/video files in `public/images/`.
2. Do not create section-based media folders.
3. Keep filenames exact (case-sensitive).

## Required Media Count

- Total files: 20
- Images/logos/icons: 19
- Videos: 1

See `IMAGE_UPLOAD_GUIDE.md` for the complete filename list.
